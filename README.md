# 🛒 SkullFace Loja — E-commerce em Produção

Aplicação real de e-commerce desenvolvida para um estúdio de tatuagem, com backend em Django, banco de dados PostgreSQL e deploy completo em produção com domínio próprio.

🔗 Acesse o sistema:
[https://www.skullfacetattoo.com.br](https://www.skullfacetattoo.com.br)

🚀 Sobre o Projeto
Este projeto é uma aplicação funcional em produção, utilizada para exibição e gerenciamento de produtos de um estúdio de tatuagem.

Foi desenvolvido com foco em construir um sistema completo, desde a estrutura do backend até a publicação em ambiente real, incluindo configuração de domínio, banco de dados e armazenamento de imagens.

🧰 Tecnologias Utilizadas
Python / Django
PostgreSQL
Cloudinary (armazenamento de imagens)
HTML / CSS
Railway (deploy e infraestrutura)
⚙️ Funcionalidades
Cadastro e gerenciamento de produtos
Upload de imagens com Cloudinary
Exibição de produtos para usuários
Painel administrativo (Django Admin)
Integração com banco de dados PostgreSQL
Sistema publicado em ambiente de produção
🚀 Deploy
A aplicação está em produção com:

Backend Django hospedado no Railway
Banco de dados PostgreSQL remoto
Domínio personalizado configurado
HTTPS ativo (SSL)
Integração com serviços externos (Cloudinary)
🎯 Objetivo
Desenvolver uma aplicação completa, abordando:

Backend com Django
Integração com banco de dados em produção
Estruturação de um sistema real
Processo completo de deploy
